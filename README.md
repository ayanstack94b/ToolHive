## 🧠 Project Overview

ToolHive is a simple product browsing and cart management application. Users can view different tools, check their features, and add them to a cart. The application provides real-time feedback and allows users to manage their selections easily.

---

## ⚙️ Tech Stack

* React.js
* Tailwind CSS
* DaisyUI
* React Toastify

---

## ✨ Features

### 🧭 Navbar

* Designed based on Figma
* Displays cart icon
* Shows total selected product count

---

### 🎯 Banner

* Includes heading, description, and image
* Contains call-to-action buttons

---

### 📊 Stats Section

* UI designed according to Figma

---

### 🔄 Product & Cart Toggle

* Toggle between **Products** and **Cart**
* Default view: Products
* Cart shows empty state when no items are selected

---

### 📦 Product Data

* 6–10 products loaded from JSON
* Each product includes:

  * id
  * name
  * description
  * price
  * billing period (one-time / monthly / yearly)
  * tag & tagType (popular, new, best seller)
  * features list
  * icon

---

### 🃏 Product Cards

* Displayed in a 3-column layout
* Each card contains:

  * Name
  * Description
  * Price
  * Period
  * Tag
  * Features
  * Icon
  * Buy Now button

---

### 🛒 Cart Functionality

* Add products to cart
* Remove individual products
* Display total selected items
* Show total price
* "Proceed to Checkout" button:

  * Clears all cart items

---

### 🔔 Notifications

* Implemented using React Toastify
* Alerts for:

  * Add to cart
  * Remove item
  * Checkout

---

### 🪜 Steps Section

* Designed according to Figma

---

### 💰 Pricing Section

* Figma-based layout

---

### 🦶 Footer

* Structured based on Figma design

---

### 📱 Responsive Design

* Fully responsive across mobile, tablet, and desktop

## 🧪 How to Run Locally

1. Clone the repository:

   ```bash
   git clone <your-repo-link>
   ```

2. Navigate to project folder:

   ```bash
   cd toolhive
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the app:

   ```bash
   npm run dev
   ```
